
# HabaHaba (HABA) - Decentralized Utility Network

A memory-hard, egalitarian social utility network built on Polygon, optimized for the Kenyan economy.

## Technical Core
- **Engine:** Rust (Security & Performance)
- **Network:** Polygon Mainnet (Low Gas)
- **Infrastructure:** Netlify (Frontend) + Supabase (Off-chain Data/RLS)
- **Security:** OWASP Top 10 Compliant + MPC Wallet recovery

## Quick Start
1. `git clone <repo-url>`
2. `cargo build --release` (For the Miner)
3. `npm install` (For the Dashboard)

## Security Note
This project utilizes ZK-Rollup logic for privacy. Do not share your `.env` files.
